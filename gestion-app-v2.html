Guía rápida de configuración

Sigue los pasos a continuación para obtener tus credenciales. Una vez guardadas, la app estará lista. Los datos se guardan en el navegador (localStorage).

🍃 MongoDB Atlas — Conexión via Backend Local

  1. Entra a mongodb.com/atlas → crea una cuenta gratuita
  2. Crea un Cluster M0 (gratis) → elige región más cercana (ej: São Paulo)
  3. En Database Access → crea un usuario con password. Guárdalos.
  4. En Network Access → agrega IP 0.0.0.0/0 (permite todo)
  5. En tu cluster → Connect → Drivers → copia el SRV string y pégalo en el archivo .env
  6. Ejecuta el backend: node server.js y deja la terminal abierta
  7. La URL del backend por defecto es http://localhost:3001

Si desplegas en Render u otro servicio, pon aquí la URL pública.

📧 EmailJS — Pasos para configurar

  1. Ve a emailjs.com → crea cuenta gratuita (200 emails/mes)
  2. En Email Services → Add Service → elige Gmail u otro → conecta tu correo
  3. Copia el Service ID
  4. En Email Templates → crea un template. Usa estas variables:
    {{to_email}} {{to_name}} {{invoice_html}} {{invoice_number}}
  5. Copia el Template ID
  6. En Account → copia tu Public Key
